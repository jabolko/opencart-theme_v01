Product Card v2 — Secondhand-Optimized

5 variations designed specifically for a used children's clothing store. Each card uses the same elements (image, manufacturer, name, price, fav, cart) plus secondhand-specific elements: condition badge, size chip, savings indicator, trust signals.

Research: How secondhand shoppers behave differently
  • Condition is king: Product condition is the #1 trust driver for secondhand purchases (ScienceDirect, 2024). It must be visible early — ideally before clicking into the product page.
  • Size = first filter: 73% of apparel shoppers filter by size first (Baymard). For kids' clothes, size matters even more because children grow fast — parents need instant size visibility.
  • Value perception drives conversion: 68% of secondhand shoppers are motivated by savings vs retail price (thredUP 2024 Resale Report). Showing original price + savings % converts better than price alone.
  • More browsing, less buying per visit: Secondhand shoppers view 3x more product pages per session than new-retail shoppers. Cards should be info-dense to reduce unnecessary clicks.
  • Trust before cart: Used items have 60-70% higher return rates when condition wasn't clear upfront. Encouraging product page visits (vs quick-add) reduces returns.
  • thredUP pattern: Condition badge + size on image overlay, brand prominent, savings %, fav on image, no cart on card (drives to product page for inspection). Clean, editorial feel.
  • Vinted pattern: Compact, info-dense cards. Brand + size + condition + price all visible without clicking. Fav as primary action.

thredUP A — Condition + size on image, no cart on card

thredUP's core pattern. Condition badge + size chip sit on the image. Body is brand + name + price with savings. Reduces returns by encouraging full inspection.
